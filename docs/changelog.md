# 📅 更新日志 (Changelog)

这里记录了《雪之恋》项目的开发历程、版本迭代与重要里程碑。

## 2025-11-24

### 🚀 开源与规范化

- **文档体系建立**：正式发布了项目的 Wiki 文档，包括角色档案、脚本规范及AI工作流，方便团队协作。
- **代码开源**：整理项目工程文件，清除敏感数据，正式将代码库开源。
- **规范确立**：制定了详细的文件命名规范（如 `bg playground.jpg`）和代码注释标准。

## 2025-11-23 (v0.5.0)

### 🔄 剧情重构与第五章实装

- **剧情重构**：对原有剧本进行了深度优化，调整了对话节奏，增加了更多心理描写。
- **新章节完成**：完成了**第五章：大学生活的“真实”面貌**的全部制作。
  - 新增场景：高数课、抢食堂大战、图书馆偶遇。
  - 新增角色：实装了**白墨萱**（结构力学魔女）和**高数老师**的立绘与剧情。
- **特效升级**：
  - 实装了 Q 版追逐战动画 (`sd_suzhi_run`)。
  - 添加了草莓牛奶喷溅的慢动作演出 (`slow_motion_pan`)。
  - 增加了手机短信 (NVL) 模式的演出效果。

## 2025-11-22 (v0.1.0)

### ✨ 核心玩法与前四章

- **内容制作**：完成了**第一章至第四章**（入学篇、军训篇）的完整流程制作。
  - 实现了军训踢正步、倒水风波、阅兵典礼等核心情节。
- **基础框架搭建**：
  - 在 `script.rpy` 中定义了所有主要角色 (`m`, `s`, `l`, `e` 等) 的变量与颜色。
  - 定义了基础转场特效 (`vpunch`, `flash`, `soft_shake`)。
  - 配置了背景音乐 (BGM) 和音效 (SE) 的播放逻辑。
- **美术资源**：导入了首批 AI 生成的背景图（操场、宿舍）和角色立绘。

## 2025-11-21 (Project Start)

### 🎉 项目启动

- **立项**：《雪之恋》正式立项。
- **技术选型**：确定使用 **Ren'Py** 引擎进行开发，采用 **Stable Diffusion** 生成美术素材。
- **前期筹备**：
  - 确定了以“长春工程学院”为现实原型的世界观设定。
  - 确立了男女主角（木子米、素织）及室友（凌宁）的基础人设。
  - 收集了第一批校园实景照片素材。
