# 👓 木子米 (Muzimi)

> “这就是‘自由’的味道啊！”

## 🎭 人设基础

- **身份**：土木工程系新生，玩家的第一人称视角 (POV)。
- **性格**：随和、有些大大咧咧、但在关键时刻很细心（如帮保洁阿姨倒垃圾），吐槽役担当。
- **外貌特征**：
  - **眼镜**：本体（被教官多次点名“那个戴眼镜的”）。
  - **体型**：瘦高，有些书卷气。

## 🎨 AI 绘画咒语 (Prompts)

虽然游戏中很少出现男主的正面立绘，但在绘制 **CG**（如牵手、背影、吃手抓饼）时需要用到。

**基础特征 (Base):**

```
1boy, solo, black hair, short hair, glasses, handsome, average build, (anime style:1.2)
```

**Q版形象 (Chibi):** 用于第五章的“抱头鼠窜”动画。

```
chibi, 1boy, running, panic expression, sweat, simple background, white background
```

## 🖼️ 特殊素材

木子米主要通过以下方式出现在游戏中：

1. **Q版小人**：
   - 代码：`show sd_muzimi_run`
   - 场景：第五章被素织追打。
2. **微信头像**：
   - 代码：`m_phone` 对话时自动显示。
   - 文件：`images/misc/muzimi_icon.png`
3. **CG 插画**：
   - `cg eye_contact` (军训对视)
   - `cg eating_pancakes` (后巷吃手抓饼)
   - `cg cleaning_stairs` (帮阿姨倒垃圾 - 背影)
