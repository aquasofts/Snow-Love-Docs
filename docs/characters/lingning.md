# ✨ 凌宁 (Lingning)

> “智者不入爱河，贵族不抢饭桌。”

## 🎭 人设基础

- **身份**：木子米的室友。
- **性格**：中二病、贵族包袱极重、在关键时刻很讲义气。
- **外貌特征**：
  - **皮肤**：白白净净（被保洁大妈认证）。
  - **气质**：优雅（自称），经常摆出夸张的姿势（如 `casual pose`）。
  - **行为**：会敷面膜，会跳机械舞。

## 🎨 AI 绘画咒语 (Prompts)

凌宁的风格要比普通男大学生更“精致”和“浮夸”。

**基础特征 (Base):**

```
1boy, solo, blonde hair (or light brown hair), fair skin, bishounen, elegant pose, sparkling atmosphere, (anime style:1.2)
```

**表情差分 (Expressions):**

- **优雅 (Pose)**: `hand on chin, confident smile, closed eyes`
- **虚弱 (Weak)**: `exhausted, pale, sweating, dying` (军训时)
- **震惊 (Surprised)**: `open mouth, shock, comedic shock` (看到木子米被追杀时)
